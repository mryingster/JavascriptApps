Medical Screening Tests

Below is a small collection of tests used to screen for different sensory issues. Each page contains information on the test and how to interpret results.

Disclaimer

These tests are for novelty use only, and should not be used to diagnose any potential issues. If you suspect you have an issue with your eyesight or hearing, please consult a medical professional.

  • Cone Sensitivity Screening - This screening test checks for responsiveness of the cones that are sensitive red, green, and blue light individually.
  • Peripheral Vision Screening - This screening test checks for the eye's senstivity in the peripheral visual field.
  • Ishihara Plates - This tests uses colored dots to draw numerals. The inability to distinguish the numbers could be indicative of a red/green color blindness.
  • Hearing Test - This tests the sensitivity of the user's hearing in the rang eof 250 to 8000 Hz.
