Hearing Threshold Test

Instructions

For best results, make sure you are in a quiet environment with over-the-ear headphones, and your system volume set to 50%. Once the test begins, different frequencies will be played in the left or right ears randomly. If you hear a tone, indicate which ear you heard it in using the buttons on screen, or the left and right arrows. When the test is complete, a graph will show your results.

Description

This pure sine tone test measures the user's response to various frequencies at different amplitudes with the goal of detecting any deficiencies in the normal range of human hearing. This test is NOT calibrated in any way, so the results are basically meaningless, though it may show relatively weak areas for a user's hearing.

Disclaimer

This is for novelty use only and not meant to be used as a medical diagnostic tool.
