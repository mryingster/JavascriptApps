Eye Screening Tests

Below is a small collection of tests used to screen for different aspects of an eye's health. Each page contains information on the test, what it tests, and how to interpret results.

  • Cone Sensitivity Screening - This screening test checks for responsiveness of the cones that are sensitive red, green, and blue light individually.
  • Peripheral Vision Screening - This screening test checks for the eye's senstivity in the peripheral visual field.
  • Color Blindness Screening - This tests uses colored dots to draw numerals. The inability to distinguish the numbers could be indicative of a red/green color blindness.

Disclaimer

The tests presented here are meant to be used for screening purposes only, and not for any type of diagnosis. If you suspect you have an issue with your eyesight, please cconsult your optometrist.
